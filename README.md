#  Curiosidades sobre o Android

Um projecto front-end desenvolvido para praticar **HTML5** e **CSS3**, com foco em **layout responsivo** e boas práticas de desenvolvimento web.

O site apresenta um breve conteúdo sobre a história do Android, utilizando uma interface simples, agradável e adaptável a diferentes tamanhos de ecrã.

##  Tecnologias utilizadas

- HTML5
- CSS3
- Design Responsivo
- Google Fonts

##  Funcionalidades

- Interface totalmente responsiva
- Layout adaptado para computadores, tablets e smartphones
- Estrutura semântica em HTML
- Estilização moderna com CSS

##  Objectivo

Este projecto foi desenvolvido com fins educativos para colocar em prática conceitos de HTML e CSS, especialmente responsividade e organização de layouts.
